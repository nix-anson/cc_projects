The API documentation is auto-generated by drf-spectacular.

**View in browser** (server must be running):
```
http://127.0.0.1:8000/api/schema/swagger-ui/
```

**Generate static schema file**:

```bash
uv run python manage.py spectacular --color --file schema.yml
```

**Available API endpoints**:
- `GET /api/politicians/` — List politicians with filtering
- `GET /api/politicians/{id}/` — Politician detail with votes and decisions
- `GET /api/politicians/{id}/votes/` — All votes for a politician
- `GET /api/politicians/{id}/funding/` — Campaign finance for a politician
- `GET /api/votes/` — Vote records with filtering
- `GET /api/decisions/` — Decision records with filtering
- `GET /api/affected-groups/` — Affected group categories
- `GET /api/funding/` — Campaign finance records
- `GET /api/sources/` — Data sources

**Filter examples**:
```
/api/politicians/?state=CA&party=D
/api/politicians/?search=pelosi
/api/votes/?vote_choice=Nay&congress_session=118
/api/decisions/?decision_type=executive_order
/api/funding/?cycle_year=2024&min_amount=100000
```
